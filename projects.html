syntheseyeze

- in progress

A software instrument controlled primarily by eye-movement. Proof-of-concept created for an arts course project at the University of Sydney using WebGazer.js and Tone.js, future work to be done using the Tobii Eye Tracker. Aimed at facilitating the musical expression for those unable to by traditional instruments and means.

long short-term sampler

- in progress

A machine learning based audio sampler with the end goal of producing novel sounds for use in music production.

the schizophoner

A simple Android application created for an arts course project exploring schizophonia (the split of sounds and their electroacoustic reproduction). Following calibration, sound events above ambient room volume (plus threshold) are buffered in a queue/random order and then played back later replacing current audio events.

artificial synesthesia network

A convolutional neural network for interpreting images musically written using Python and Keras. Developed as part of a directed study guided by Prof. Abram Hindle at the University of Alberta.

review board

A modified design of the diff viewer interface of the open-source project, Review Board, with the goal of being more mobile-friendly, as part of the UCOSP program. Included weekly development meetings, as well as several project demos. Video of final demo available here.

paladins auto login

A now-defunct Python script created to automate the process of logging in to the videogame Paladins to retrieve daily rewards.
