syntheseyezer

- under construction

A software instrument controlled primarily by eye-movement. Proof-of-concept created for an arts course project at the University of Sydney using WebGazer.js and Tone.js, future work to be done using the Tobii Eye Tracker.

the schizophoner

A simple Android application created for an arts course project exploring "the split of sounds and their events." Once room level is calibrated, can be used to record and play audio simulatenously. Sound events above a certain threshold can be buffered in a queue/random order and then played back later replacing current events.

artificial synesthesia network

A neural network written using Python and Keras for interpreting images musically. Developed as part of a directed study guided by Prof. Abram Hindle at the University of Alberta.

review board

Modified the design of a specific section of the interface of an open-source project, Review Board, to be more mobile-friendly, as part of the UCOSP program. Included weekly development meetings, as well as several project demos. Video of final demo available here.

paladins auto login

A now defunct Python script created to automate the process of logging in to the videogame Paladins to retrieve daily rewards.
